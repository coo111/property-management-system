# 小区物业管理系统

## 一、介绍

运行环境:idea或eclipse 数据库:mysql

开发语言：java

基于springboot + mybatis + thymeleaf + Mysql的物业管理系统

技术栈：springboot + mybatis + thymeleaf + Mysql

角色：管理员和用户

控制台、数据库、楼栋管理、单元管理、房屋管理、车位管理、缴费类型、缴费管理、公告管理、维修管理、投诉管理、用户管理
